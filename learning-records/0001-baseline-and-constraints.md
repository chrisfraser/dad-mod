# Baseline: profile, goals, and hard constraints established

Established at workspace start (pre-teaching), from the learner's own statements.

- **Profile:** 40yo male, 105 kg, 185 cm, BMI ~30.7, reasonably fit/active, **new to structured
  resistance training**. Sits in the general-adult evidence base, not the elderly one — standard
  hypertrophy principles apply with only minor age hedges.
- **Goals:** build/maintain muscle mass + improve metabolic health. Strong recomposition candidate
  (untrained + higher body fat) → expect fast, front-loaded early progress.
- **Capacity:** 3 days/week, ~45 min/session. Equipment: basic free weights + a few machines (TBC).
- **Medical constraint (load-bearing for all future sessions):** previously fractured **scaphoid**.
  Wrist cannot bear load in extension; flat-palm push-ups out. All exercise selection must favour
  neutral-wrist / machine / strap-assisted / fist-supported variations. This is permanent design input.

## Implications
- Start at the **low end of volume** (~8–12 hard sets/muscle/week) and progress slowly.
- Default to **full-body 3×/week** with **double progression**.
- Every pressing/pulling/squatting choice is filtered through the wrist constraint.
- Prior knowledge of training theory: assume **near zero** — teach principles before nuance.
- Open question: no recent bloodwork/check-up → flag baseline metabolic testing.
