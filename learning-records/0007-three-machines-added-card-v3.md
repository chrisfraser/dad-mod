# Three machines added → Program Card v3

Session 11 (2026-07-22). Chris reported the gym added three machines. No lift removed — all three
*add* wrist-safe options and progression headroom, so the A/B structure and volume are unchanged.

**New equipment**
- **Technogym multi press** — adjustable chest + shoulder press, **neutral-grip handles**, weight stack.
  Verified via Technogym product pages: independent movement arms + neutral grips "for users with limited
  shoulder mobility." Wrist-safe (neutral grip = no loaded extension) **and** on a stack → escapes the 30 kg DB cap.
- **Vision functional trainer** — dual adjustable-pulley cable machine (D-handles, V-bar, triceps rope, curl bar).
- **Technogym hip abductor** — glute-medius / outer-hip machine.

**Design decisions (Program Card v3)**
- **Multi press = the alternate on both press slots** (A2 chest, B2 shoulder) and the go-to once DBs feel light.
  Kept DB pressing *primary* (stabiliser demand + real-world carryover); machine is the wrist-safe, uncapped option,
  not a replacement. This reverses the v2 "machine chest/shoulder press are gone" call.
- **Cable row = alternate on both row slots** (A4, B4) — loads past the 30 kg DB ceiling, neutral V-bar = wrist-safe.
- **Cable pull-through = grip-/wrist-free RDL fallback** (B1 alt) for days grip taps out before the hamstrings.
- **Face pulls** added as an optional accessory to Workout A (rope, 2×12–15) — rear delts + shoulder health for a
  desk-bound lifter; wrist neutral.
- **Pallof press** (already the plank alt) is now a *real* loaded cable option, not a bodyweight stand-in.
- **Hip abductor** = optional accessory in Workout B (2×12–15). Deliberately not a core slot — nice-to-have for
  hip stability, kept off the critical 45-min path.

**Wrist rule holds:** every addition keeps the wrist neutral. Multi-press neutral handles and all cable handles
load through a straight wrist (press/pull, no extension) — same principle as the neutral-grip DB press and hanging
barbell already cleared in [0006].
