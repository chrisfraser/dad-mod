# Two workouts → three full-body sessions → Program Card v4

Session 11 (2026-07-22, cont'd). Chris asked to move from the alternating A/B rotation to **three separate
sessions**, still 3 days/week, using the new machines. Delivered as v4.

**Key decision: three FULL-BODY sessions, not a body-part split.**
At 3 days/week a classic split (chest / back / legs day) hits each muscle only **1×/week**. Three full-body
sessions keep every muscle at **2–3×/week** — the frequency that maximises weekly growth for a beginner, and the
mission's stated target. So A/B/C rotate the *lifts*, not the *muscles*. Rationale added as a callout in §1.

**The three sessions** (slots 1–3 = big lifts first):
- **A — squat + horizontal push/pull:** goblet squat · DB bench (alt multi press) · cable row · leg curl ⟂ face pull · plank.
- **B — hinge + overhead push/pull:** barbell RDL (alt pull-through) · leg press · DB shoulder press (alt multi press) · lat pulldown · dead bug ⟂ hip abductor.
- **C — single-leg + cable/machine day:** DB split squat · multi press (chest, main press) · cable row · cable pull-through · hip abductor ⟂ Pallof press. Showcases the new kit.

**Weekly frequency check (all ≥2×):** quads A/B/C=3× · hams A/B/C=3× · chest A+C=2× · shoulders B=1× direct
(+ indirect from all pressing) · rows A+C=2×, pulldown B=1× (back total 3×) · rear delt A=1× · glute-med B+C=2× ·
core 3×. Weekly hard sets stay in the 8–12 target band per muscle. ~6 exercises / ~15–16 sets each → ~45–55 min
with accessory supersets, matching the prior card's length.

**Wrist rules unchanged** — every lift still loads a neutral wrist (multi-press neutral handles, cables pull/press
from a straight wrist, hanging RDL, goblet/bells). See [[0006-gym-equipment-confirmed-card-v2]] for the underlying
"loaded wrist extension is the enemy" rule and [[0007-three-machines-added-card-v3]] for the machine additions.
